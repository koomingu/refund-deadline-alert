# 설문 응답을 Google Sheets에 저장하기

1. 새 Google Sheets를 만들고, **확장 프로그램 → Apps Script**를 엽니다.
2. 기본 파일 내용을 지우고 `Code.gs`의 내용을 붙여넣은 뒤 저장합니다.
3. **배포 → 새 배포 → 유형: 웹 앱**을 선택합니다.
4. 실행 계정은 본인, 액세스 권한은 **모든 사용자**로 설정하고 배포합니다.
5. 권한을 승인한 뒤 생성된 **웹 앱 URL**을 복사합니다.
6. URL을 Codex에 보내면 랜딩 페이지가 해당 스프레드시트에 응답을 저장하도록 연결합니다.

`응답` 시트가 자동으로 만들어지며, 한 행에 이메일·유입 경로·설문 응답·진단 결과가 저장됩니다.

## 설문 항목을 추가했을 때

`Code.gs`를 최신 내용으로 교체한 뒤, **배포 → 배포 관리 → 수정 → 새 버전**을 선택해 다시 배포하세요. 기존 웹 앱 URL은 그대로 사용할 수 있습니다.
